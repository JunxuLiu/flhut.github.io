Introduction

FL Hut is an open-source deep learning library written in Python, specialized in federated learning .

  • federated sgd,
  • federated averaging,

Algorithms

SPMF offers implementations of the following data mining algorithms.

Federated Learning

These algorithms discover federated learning w.r.t heterogeneous data/system settings. For a good overview of federated learning, please read this survey paper.

  • algorithms for mining sequential patterns (subsequences that appear in many sequences) of a sequence database
    • the CM-SPADE algorithm (Fournier-Viger et al, 2014, powerpoint)
  • algorithms for personalized federated learning
    • the CM-SPADE algorithm (Fournier-Viger et al, 2014, powerpoint)

These algorithms discover privacy-preserving federated learning.

  • algorithms for federated learning with differential privacy
    • the SPAM algorithm (Ayres et al., 2002)
  • algorithms for federated learning with personalized differential privacy
    • the SPAM algorithm (Ayres et al., 2002)

The current version is v0.1 and was released the 23rd Oct. 2021.
